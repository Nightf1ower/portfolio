# Изображения для секции Works

Секция Works ожидает изображения в этой папке. Текущие пути указаны в `src/works.js`:

- `brand-identity.jpg`
- `editorial-campaign.jpg`
- `packaging-concept.jpg`
- `social-media-kit.jpg`
- `poster-series.jpg`
- `web-visuals.jpg`

Если у ваших файлов другие названия, обновите поле `image` у соответствующего проекта в `src/works.js`.
